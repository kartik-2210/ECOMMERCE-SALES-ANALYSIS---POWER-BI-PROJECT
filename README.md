# Madhav Ecommerce Sales Dashboard – Power BI Project

## Project Overview
This project focuses on building an interactive Power BI Dashboard for Madhav Store to help track, analyze, and visualize online sales performance across India.

The dashboard provides meaningful business insights related to:
- Sales performance
- Profit trends
- Customer behavior
- Product category performance
- State-wise sales analysis
- Payment mode analysis

The main goal of this project is to transform raw sales data into an easy-to-understand and visually interactive dashboard that supports better business decision-making.

# Objective
The owner of Madhav Store wanted a dashboard that could:
- Track online sales across different states in India
- Analyze monthly profit and revenue trends
- Identify top-performing product categories and sub-categories
- Understand customer purchasing behavior
- Monitor payment method usage
- Enable interactive filtering for better analysis

---

# Tools & Technologies Used
- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- Interactive Visualizations

---

# Project Workflow

## 1. Data Collection & Import
- Imported raw ecommerce sales dataset into Power BI.
- Connected and loaded the data for further processing.

## 2. Data Cleaning & Transformation
Performed data transformation using Power Query, including:
- Handling missing/null values
- Removing duplicates
- Correcting data types
- Creating calculated columns
- Formatting data for analysis

## 3. Data Modeling
- Established relationships between tables
- Structured the dataset for efficient reporting and filtering

## 4. DAX Calculations
Created measures and calculated columns using DAX for:
- Total Sales Amount
- Total Profit
- Average Order Value (AOV)
- Quantity Sold
- Profit Analysis

---

# Dashboard Features

## Interactive Dashboard
The dashboard is fully interactive and allows users to:
- Filter data by Quarter
- Filter sales by State
- Interact dynamically with charts and visuals

---

# Visualizations Used

The dashboard includes multiple visualizations such as:
- Cards → Display KPIs like Total Sales, Profit, Quantity, and AOV
- Bar Charts → State-wise and sub-category analysis
- Donut/Pie Charts → Category-wise and payment-mode analysis
- Area/Line Charts → Monthly profit trends
- Slicers → Quarter and state filtering
- Interactive Filters → Dynamic dashboard exploration

---

# Key Insights from Dashboard

- Maharashtra generated the highest sales amount.
- Clothing category contributed the largest share of quantity sold.
- COD (Cash on Delivery) was the most preferred payment mode.
- Printers and Bookcases generated the highest profit among sub-categories.
- Monthly profit showed fluctuations with strong performance during certain months.

# Business Questions Answered

This dashboard helps answer important business questions such as:

1. Which states generate the highest sales revenue?
2. Which product categories perform best?
3. Which sub-categories contribute the most profit?
4. What are the monthly profit trends?

6. Who are the top customers by sales amount?
7. Which quarter performs best in terms of sales and profit?
8. How does quantity sold vary across product categories?
9. Which months experience profit decline?
10. What is the overall Average Order Value (AOV)?
